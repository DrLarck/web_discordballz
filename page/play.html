Play

To play the game, you must Add the game to your server.

Note

To do so, you will need a Discord account.

Once you have added the game to your server, you will be able to use its commands.

Prefix

As the game is considered as a bot user, you will need to use a certain syntax to interact with it.

This syntax includes in particular the use of a set of prefixes. If you do not use one of those, the game won't be able to understand your request !

Here is the set of prefixes the game understand :

• d!command

• D!command

• dbcommand

• Dbcommand

Replace the command by the command name you want to use.

Example of commands usage

Here is an example of the summon command which allows us to summon a character and add it to our collection :

d!summon basic

By typing this, I tell the game I want to use the summon command with the basic parameter.

Some commands (such as summon) can do a lot of things, that is why the usage of parameters is needed to help the game understand exactly what you are trying to do with the command.
Here is what the summon command skeleton looks like :

summon {expansion} {type}

Each word between braces ({}) represents a parameter.

For example, using the command summon basic will have a different behaviour than the summon expansion multi command.
That is why the use of parameter is important.

You will find more informations about commands usage in the Guide.
