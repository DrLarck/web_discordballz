Start -

First of all, you need to invite the game to your server to play it.
To do so, click this link, choose the server where you want to invite the game to, and you are done ! It is as simple as that !

The invitation link should redirect you to this pannel.

You must know that Discord Ball Z is a text-based game which means that you need to use commands to interact with the game.

To call a command you must start your message with a prefix.

You can interact with Discord Ball Z by using one of the following prefixes :

  • d!
  • D!
  • db
  • Db

A message using game prefix (a.k.a command).

Know you know how to play, you must register yourself to use the rest of the commands.
To do so, type the command d!start in the chat as shown by the image above.

Fighter -

In Discord Ball Z you manage a Team of characters. This team can be composed from 1 to 3 characters.

Each character can be assigned to a character slot.

As you can assign maximum 3 characters to your team slots you have 3 available character slots :

  • A : Leader slot üëë
  • B : Normal slot
  • C : Normal slot

To assign a character to a specific slot you have 2 possibilities (using the fighter command) :

  • First possibility : By Global id
    The Global id method consists to pass the character's global id as parameter to the fighter command, like : d!fighter set a 6
