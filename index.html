Welcome

Hello and welcome to the Discord Ball Z Official website !

What will I be able to find in here ?

You will be able to find many usefull informations such as :
• A list of character showing their stats, abilities and a note on how to use them well.
• The latest patchnote and a list of all the archived ones.
• Some tutorials (video or text)
• A Things to come section.
• And a lot more ...

Am I forced to use this website to play the game ?

No, absolutely not. This website is used as a tool to allow you to track what's coming up next or what happened to a character/feature during a previous patch.
Also, if you are a beginner this website might be usefull for you, as it contains a lot of details about How to play ? and other beginners' stuffs like that.

Note

Please note that most of the time you ask a question about "How to do this", "How to ... you will be redirected to this website.
So please take some time to read and understand its content.
