Patch 3.0.3

Summary

  • Character balance changes
  • New Feature
  • Bug Fixes
  • Other

Character balance changes

New feature

  • Add new combat system :
    • The new combat system is smoother than the previous one. The visual pollution has been removed, meaning that combats are going to be less "spammy".

      Moreover, the new combat system allows PvP, you can test the new PvP feature by using the d!pvp [@user] command.

Note

Note that you can still flee a fight or check a fighter by typing flee or check [fighter index] in the chat during a combat.

  • Add the d!pvp [@user] command : This command allows you to start a fight against another player.

Bug fixes

Other

  • Add PvP help panel.
  • Rename the d!fighter command group to d!team.
  • Typing d!team will now display your team.

Note

We have decided to change the command group name to team because we think that it is more intuitive.
The other commands that you were using with the old d!fighter command group such as d!fighter set [a/b/c] are still available. You'll just need to replace the d!fighter by d!team.
