Patch 3.0.3.134

Summary

  • Character balance changes
  • New Feature
  • Bug Fixes
  • Other

Character balance changes

  • Broly #6 :
    • Add Powered Shell ability :
      Gains 10 % damage reduction for 5 turns.

    • Add Eraser Canon ability :
      Inflicts 15 % of your Ki power as damage to all the enemies and Stun them for 1 turn.

New feature

  • The d!show [id] [level] now displays the passive and leader skills of the character.

Bug fixes

  • Fix a bug that was failing the Summon feature to display rarity color properly.
  • The stat bonuses are now calculated properly.
  • Fix a bug that was failing the CPU player to pick a fighter properly if one of its fighter were under a stun effect.
  • Fix a bug that was failing the d!show [id] [level] command to display character properly.
  • Fix a bug that was failing the Trainer to give XP to the player's team at the end of the combat.
  • Fix a bug that was failing the Red Saibaiman's passive to be applied properly.
  • Fix a bug that was making the CPU player target its own team.

Other

  • Add a global stun effect.
  • The stat comparison is now handled properly by the check [index] command in combat.
  • Add image to all characters.
  • Add thumbnail to all characters.
  • Add icon to all characters.
  • Fix a message in the fight checker referencing the d!fighter command that no longer exists.
  • The passive skills trigger order is now properly handled by the new combat system.
