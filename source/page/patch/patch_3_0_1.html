Patch 3.0.1

Summary

  • Character balance changes
  • New Feature
  • Bug Fixes
  • Other

Character balance changes

  • Blue Saibaiman
    • Abilities :
      • Remove : Replenish the ranks
  • Pilaf Machine
    • Passive :
      • Add : Triple pilots : Generate 2 stacks of Triple pilots bonus for the caster.
    • Abilities :
      • Add : Rolling smash : This ability deals 150 % of your physical damage and has a 4 turns cooldown.
      • Add : Pilaf barrier : This ability gives a bonus to the caster which increases its armor by 50 % for 1 turn.
        Moreover, the caster is considered as Defending for the effect duration.
        3 turns cooldown.
      • Add : Triple pilots : This ability consums one charge of Triple pilots and allow the unit to recover 30 % of its Maximum health.

Developer comment

"We have decided to remove the Replenish the rank ability pending its rework."

New feature

  • Fighter selection :

    We have changed the way of setting a character as fighter.

    Before, you were forced to know the character's unique id to define it as fighter.

    Now, you can type, for example : fighter set a {id} to open a box page which will display a set of characters with the same id as the passed one.
    Then, you just have to type the character's index number and let the game assign it automatically.

Developer comment

"We think that it is the easiest way to define a fighter.
You no longer need to open your box separately to check your character's id one by one.
It is also more intuitive."

    The old method also works : fighter set a {unique id}

  • Team management :

    The update also implements a Team Management system which allows you to remove, add or replace one of your fighters easier.

    The commands are the following :
    • set {slot} {id} : This command allows you to add or replace a fighter at the passed slot by the character passed as id.
    • remove {slot} : Set the passed slot to None (Which basically means "empty")
    If you just want to display your team, you can use the command fighter team.

    Originally the leader of your team is the fighter defined at slot A.

Bug fixes

  • Fixed a bug that was failing the Box to count the page amount correctly.
  • Fixed a bug that was failing some abilities to add a stack of Acid (DOT) properly.

Other

  • Profile : The profile command has been removed and will be added for the next patch.
