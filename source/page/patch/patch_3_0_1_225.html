Patch 3.0.1.225

Summary

  • Character balance changes
  • New Feature
  • Bug Fixes

Character balance changes

  • Goten
    • Add skeleton.
  • Turles
    • Add skeleton.
  • Launch
    • Add skeleton.
  • Piccolo
    • Abilities :
      • Add : Arm stretch :
        Performs a sequence attack against any enemy.

      • Add : Special Beam Cannon :
        Inflicts 50 % of the unit's Ki power as True damage.

New feature

  • Add cancel command.
  • Add cancel fight command.

Note

We have added the cancel command to avoid the player to get stuck in the middle of a blocking action.
This command can be used as any other command, here's a quick example : d!cancel fight
We will add other functionality to this command in the future.

Bug fixes

  • Fix a bug that was failing the start command to work properly if the player's name contains the character : '.
  • Fix a bug that was making the fight system to crash during a fight.

Other

  • The flee combat option is not case sensitive anymore.
  • The opponent team in the train now contains the 3 Saibaiman.

Note

We have added the 3 Saibaiman to the train feature to test out the AOE abilities.
